## 角色体系

| 级别 | key | 中文名 | 权限要点 |
|---|---|---|---|
| 0 | `volunteer` | 常驻志愿者 | 查看、提交、发帖、抢票 |
| 1 | `dept_head` | 部门负责人 | + 发布本部门任务/公告/票务、生成本部门志愿者邀请码 |
| 2 | `presidium` | 主席团成员 | + 跨部门发布、查看所有成员 |
| 3 | `president` | 主席 | + 人员裁撤、部门调动 |
| 3 | `teacher` | 老师 | + 人员裁撤、部门调动（同主席） |

## 使用说明

- **学生**：在「学生登录」页输入姓名 + 学号 + 对应邀请码
- **教师**：在「教师入口」页输入姓名 + 工号 + `TEACHER0X` 邀请码
- **登录方式**：注册后使用 `学号@stuunion.org` + 密码登录
