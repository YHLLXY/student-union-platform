## 三、🟡 功能缺口

### 3.1 全平台共性缺口

| 缺口 | 影响 |
|------|------|
| 全平台无批量操作 | 不能批量分配任务、批量生成邀请码 |
| 无数据导出（Excel/CSV） | 成员名单、任务汇总只能在线看 |
| 无新人引导流程 | 新志愿者登录后面对空仪表盘 |
| 全局搜索仅覆盖4张表 | tickets、school_notices、guides 搜不到 |
| 无 @提及功能 | 不能跨模块标记/通知特定成员 |

### 3.2 模块级缺口

**任务管理：**
- 无搜索、筛选、批量操作
- 任务发布后不可编辑内容
- 无任务评论/讨论线程
- 无定期任务、无依赖关系

**论坛：**
- 无搜索、无分页
- 无点赞/收藏/置顶
- 纯文本输入，无富文本

**抢票：**
- 无活动详情页
- 无 QR 签到、无主办方查看名单
- 往期活动不可见

**通知：**
- 仅保留最近20条
- 无浏览器推送通知（Web Push）
- 同类通知无聚合

**登录：**
- 无密码强度检测、无验证码
- 无记住我、无会话超时处理

## 六、推荐推进路线

```

  RLS加固 → 修复月报逾期Bug → 论坛N+1 → 加LIMIT → 清理硬编码密钥

  活动签到(QR) → 任务搜索+筛选 → 数据导出 → 成员考核积分

  招新系统 → 帖子互动 → 新人引导 → 全模块搜索 → 通知扩容

阶段四（锦上添花）
  经费管理 → 定时发布 → 投票评选 → 深色模式 → AI辅助
```
