# 论坛权限与可见性改进 — 设计文档

> 状态：已确认。基于 2026-06-25 讨论。

## 一、目标

1. 控制发帖权限：志愿者不可发帖，dept_head+ 可发帖
2. 删除帖子：发帖者可自删，dept_head+ 可删本部门帖，president/teacher 可跨部门删帖
3. 部门隔离：帖子默认仅本部门可见
4. 协同部门：高级权限者（presidium+）发帖时可选择协同部门，发帖后可追加
5. 帖子展示发布者所属部门

## 二、数据库变更

```sql
ALTER TABLE forum_posts ADD COLUMN collaborating_departments TEXT[] DEFAULT '{}';
```

## 三、可见性规则

帖子对用户可见：**用户所在部门 ∈ {post.department} ∪ post.collaborating_departments**

| 发帖者角色 | 默认可见范围 | 可选协同 |
|---|---|---|
| dept_head | 仅本部门 | 不可选 |
| presidium / president / teacher | 本部门 | 可多选任意部门 |

volunteer 不可发帖。

## 四、权限矩阵

| 操作 | volunteer | dept_head | presidium | president/teacher |
|---|---|---|---|---|
| 看帖（本部门+协同） | ✅ | ✅ | ✅ | ✅ |
| 发帖 | ❌ | ✅ | ✅ | ✅ |
| 回帖 | ✅ | ✅ | ✅ | ✅ |
| 删自己帖 | ✅ | ✅ | ✅ | ✅ |
| 删同部门别人帖 | ❌ | ✅ | ✅ | ✅ |
| 跨部门删帖 | ❌ | ❌ | ❌ | ✅ |
| 选协同部门 | ❌ | ❌ | ✅ | ✅ |
| 追加协同部门 | ❌ | ❌ | ✅ | ✅ |
