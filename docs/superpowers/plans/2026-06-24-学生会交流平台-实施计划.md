# 学生会线上交流平台 — 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 从零搭建学生会内部交流平台，包含登录认证、任务管理、公告、论坛、抢票、权限管理 8 大模块，部署到 GitHub Pages。

**Architecture:** React 19 + Vite + TypeScript 前端，Ant Design 5 UI 组件库，Supabase 云后端（认证 + PostgreSQL + 实时订阅），CSS Modules 样式隔离。8 个独立模块，每个模块只有一个对外入口 `index.tsx`。

**Tech Stack:** React 19, Vite 6, TypeScript 5, Ant Design 5, React Router 7, Supabase JS Client, CSS Modules, GitHub Pages

**项目根目录:** `E:/homework/开发/Claudecode/学生会交流平台`
