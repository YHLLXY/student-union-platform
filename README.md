# 学生会线上交流平台

> 重庆邮电大学学生会内部办公与交流平台

**🌐 线上地址：https://yhllxy.github.io/student-union-platform/**

## 技术栈

React 19 · TypeScript 6 · Vite 8 · Ant Design 6 · Supabase

## 本地开发

```bash
npm install
npm run dev      # http://localhost:5173/student-union-platform/
npm run build    # 生产构建
```

## 部署

推送 `master` 分支 → GitHub Actions 自动构建 → GitHub Pages
