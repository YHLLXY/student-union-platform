## 快速开始

### 我需要什么？

- 一台能上网的设备（电脑或手机浏览器）
- 你的学号和密码（由部门负责人或管理员分配）

### 访问地址

**🔗 [yhllxy.github.io/student-union-platform](https://yhllxy.github.io/student-union-platform/)**

建议使用 **Chrome 浏览器**（电脑端、安卓端均可），体验最佳。iOS 用户使用 Safari。

## 登录与注册

### 首次注册

1. 打开平台网址，点击 **"学生（首次使用请注册）"** 标签
2. 填写你的真实姓名和学号
3. 选择你所属的部门（如"宣传部"、"体育部"等）
4. 输入部门邀请码（向部门负责人索取）
5. 设置密码 → 点击注册

> ⚠️ **注意：** 邀请码每位新成员使用一次即作废。如果提示邀请码无效，请联系部门负责人获取新的。

### 日常登录

1. 打开平台网址
2. 点击 **"学生（已注册请登录）"** 标签
3. 输入学号和密码 → 登录

### 教师入口

教师使用 **"教师（教职工登录）"** 标签登录，无需邀请码。

> 💡 忘记密码？请联系管理员重置（管理员会将新密码告知你）。

## 角色与权限

平台有 5 种角色，不同角色看到的功能和能做的操作不同：

| 角色 | 我能做什么 |
|------|------|
| 🟢 **常驻志愿者** | 查看和提交任务、看公告、逛论坛、抢票、看个人统计 |
| 🟡 **部门负责人** | 志愿者权限 + 发布任务/公告、审核任务提交、管理本部门成员 |
| 🟠 **主席团成员** | 部门负责人权限 + 跨部门查看数据 |
| 🔴 **主席 / 老师** | 最高权限 + 全校数据访问 + 成员管理 |

> 不理解某个功能为什么用不了？大概率是你的角色权限不够。可以询问部门负责人。

## 首页工作台

登录后看到的第一个页面。包含三个区域：

### 统计卡片

顶部的数字卡片展示你的任务数据——待开始、进行中、待审核、已完成各有多少。

### 活动时间线

按时间倒序展示平台最近动态：新发布的任务、新的公告等。

### 快捷操作

- **我的任务** — 跳转到任务列表，自动筛选出你的任务
- **部门公告** — 跳转到公告页面
- **月度报告**（部门负责人及以上可见）— 查看本月各部门任务完成统计和成员排行榜

## 任务管理

这是最核心的日常功能。任务的完整流程：

```
负责人发布任务 → 志愿者执行 → 提交完成 → 负责人审核 → 通过 / 打回
```

### 列表 vs 看板

页面顶部可以切换两种视图：
- **📋 列表** — 传统列表形式，适合快速浏览
- **📌 看板** — 按状态分列（待开始 / 进行中 / 待审核 / 已完成），可以 **拖拽卡片** 改变任务状态

### 搜索与筛选（v3.1 新增）

列表模式下，在状态标签页上方可以：
- 🔍 **搜索框** — 输入关键词，按任务标题实时过滤
- **优先级筛选** — 只看紧急 / 重要 / 普通的任务
- **部门筛选** — 只看某个部门的任务（跨部门管理者可见）
- **重置按钮** — 一键清除所有筛选条件

### 发布任务（部门负责人及以上）

1. 点击右上角 **"发布任务"** 按钮
2. 填写：标题（必填）、内容（支持 Markdown 格式）、优先级、截止时间、执行人（留空 = 整个部门）
3. 可选：从模板创建、添加附件、启用里程碑
4. 点击提交 → 被指派人会立刻收到通知

### 提交任务

1. 点击任务卡片打开详情
2. 在底部填写完成说明
3. 可选的交接备注（方便后续接手的同学了解情况）
4. 点击 **"提交任务"** → 状态变为"待审核"

### 审核任务（部门负责人及以上）

1. 打开已提交的任务详情
2. 查看提交记录 → 点击 **"通过"** 或 **"打回"**

### 编辑任务（v3.1 新增）

任务发布后发现写错了？点击任务详情右上角的 **"编辑"** 按钮，可以修改：
- 标题、内容、优先级、截止时间、执行人、附件

> ⚠️ 只有 **任务发布者** 或 **管理员（主席/老师）** 可以编辑。已完成的任务不可编辑。

### 模板管理（部门负责人及以上）

经常发布同类型的任务？点击 **"模板管理"** 创建任务模板，下次发布时一键填入内容和步骤清单。

## 部门公告

公告是部门内部的信息传达渠道。

### 查看公告

进入"部门公告"页面，最新公告置顶显示。点击公告卡片可查看详情。

### 已读确认

打开公告详情后，你的阅读状态会自动标记为"已读"。发布者可以点击 **"已读/未读"** 查看哪些成员还没看。

### 发布公告（部门负责人及以上）

点击 **"发布公告"**，支持：
- 普通通知：纯文字 + 附件
- 紧急通知：置顶高亮显示
- **一键转为任务**：发布公告的同时创建关联任务——适合"明天开会带材料"这类需要执行的通知

---

## 部门论坛

部门内部的讨论空间。

### 发帖

点击 **"发布帖子"**：
- 选择分类（经验分享、提问求助、通知通告、其他）
- 标题 + 内容（支持 Markdown 格式）
- 可添加附件

### 知识库模板

发帖时可以使用 Markdown 知识库模板，快速写出结构化的经验文档。

---

## 活动抢票

用于活动名额的限时发放。

### 抢票

1. 在"活动抢票"页面查看进行中的活动
2. 有余票的活动点击 **"立即抢票"**
3. 抢票成功后在我的票券中查看

### 退票

如果临时不能参加，在"我的票券"中点击 **"退票"**，票数会回流供其他人抢。

### 发布活动（部门负责人及以上）

点击 **"发布活动"**：填写活动名称、描述、总票数、截止时间等。

## 个人中心

你的个人数据总览页面，包含多个板块（通过 Tab 切换）：

| 板块 | 内容 |
|------|------|
| 📊 **个人统计** | 任务完成数、通过率、活跃度等数据 |
| 🔥 **活跃热力图** | 本学期每日任务完成情况的可视化图表 |
| 🏆 **排行榜** | 部门/全校成员的任务完成排名 |
| 📅 **日历视图** | 按月份查看任务时间分布 |
| 👥 **成员通讯录** | 查看全学生会成员列表（联系方式受权限控制） |
| 📖 **新人指南** | 部门新人入门文档 |

## 权限管理（管理者）

仅部门负责人及以上可见。

### 成员管理

- 查看本部门（或全校）所有成员列表
- **修改角色**：提升或降低成员权限
- **移除成员**：将成员移出部门
- **重置密码**：为忘记密码的成员重置密码
- **转部**：在部门之间调动成员

### 邀请码管理

- 生成新邀请码供新成员注册
- 查看每个邀请码的使用状态（可用 / 已使用 / 已停用）
- 停用未使用的邀请码

### 工作看板

查看部门/全校成员的任务统计：待办数、进行中、逾期数等。

## 通知中心

顶部导航栏右侧的 🔔 图标。

### 通知类型

| 通知 | 触发时机 |
|------|------|
| 📋 新任务指派 | 有人给你分配了任务 |
| ✅ 提交通过了 | 你提交的任务被审核通过 |
| ↩️ 提交被驳回 | 你提交的任务被打回 |
| 📢 新公告 | 你所在部门发布了新公告 |
| 💬 论坛回复 | 你的帖子有了新回复 |

### 查看通知

点击铃铛图标 → 弹出通知列表（PC）或侧滑面板（手机）。

未读通知会有红色圆点标记。点击通知可跳转到相关内容。

## PWA 安装（手机端）

平台支持以 **类原生 App** 的方式安装在手机上：

### 安装步骤（安卓 Chrome）

1. 用 Chrome 浏览器打开平台网址
2. 浏览器会自动弹出安装横幅 → 点击 **"安装"**
3. 如果没有弹出，点击右上角 **"⋮"** → **"添加到主屏幕"**
4. 桌面出现 🏛 图标 → 点击即可全屏打开，和普通 App 体验一致

### 安装步骤（iOS Safari）

1. 用 Safari 打开平台网址
2. 点击底部 **"分享"** 按钮（方框+箭头图标）
3. 滑动找到 **"添加到主屏幕"**
4. 确认后桌面出现图标

### 版本更新

平台更新后，打开 App 时底部会出现"新版本已就绪"提示，点击即可刷新到最新版。刷新后会自动弹出更新内容公告。

## 常见问题

### Q: 注册时提示"邀请码无效"？
邀请码已被使用或停用。请联系部门负责人给你一个新的邀请码。

### Q: 忘记密码怎么办？
联系部门负责人或管理员帮你重置密码。重置后的新密码由管理员告知你。

### Q: 为什么我看不到某些功能？
不同角色看到的菜单不同。比如"权限管理"只有部门负责人及以上可见。如果不确定，问你的部长。

### Q: 任务提交后还能改吗？
提交后需等待审核，不能直接修改。如果需要补充说明，可以在交接备注中追加。审核打回后可以重新提交。

### Q: 搜索不到我想找的内容？
全局搜索目前覆盖任务、公告、论坛和成员。任务列表页的搜索框可以按标题搜索任务。如果还找不到，建议检查筛选条件是否限制了范围。

### Q: 手机上使用有什么注意事项？
- 推荐使用 **Chrome 浏览器**（安卓）
- 建议安装 PWA 到桌面，体验更好
- 如果弹窗看起来奇怪（超出屏幕），刷新页面即可——最新版已修复此问题

### Q: 发现 Bug 或有功能建议？
PC 端顶部导航栏有 **🐛 反馈** 按钮，点击后填写问题和联系方式，开发团队会收到。
